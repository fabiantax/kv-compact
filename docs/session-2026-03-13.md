## Download Issues Pattern

Multiple downloads resulted in corrupted GGUF files:
- `tensor data is not within the file bounds` errors
- Network interruptions on large files (>100MB)
- Possible llama.cpp version incompatibility with newer GGUF formats

**Workarounds to try:**
1. Use smaller quantization (Q3_K_M instead of Q4_K_M) for faster downloads
2. Download via browser/IDE which may handle resuming better
3. Update llama.cpp to latest version
4. Try huggingface-cli download tool
